# Agentic_log_investigator
An agentic AI system that ingests operational logs, retrieves grounded evidence using hybrid search (BM25 + vector retrieval), investigates incidents through tool-using LLM workflows, and returns structured root-cause insights with evaluation, tracing, and observability.

****Overview****
Modern production systems generate massive volumes of heterogeneous logs, making incident investigation slow, manual, and error-prone.
Traditional dashboards answer what happened, but not why.
Agentic Log Investigator is an LLM-powered agent that:

1. Retrieves the right operational context

2. Reasons using tools, memory, and structured prompts

3. Produces grounded root-cause hypotheses

4. Provides actionable next steps

5. Enforces evaluation, observability, and cost control
